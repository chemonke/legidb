API usage

Pull JSON directly from LegiDB. Endpoints are unauthenticated and live under /api.

Method Endpoint Returns
GET /api/foods All foods with their Annex III category and linked simulants.
GET /api/foods/<id> Single food record, including category metadata and simulants.
GET /api/substances Authorised substances with CAS, FCM, EC reference, and SMILES.

Example calls

Replace localhost:5000 with your host/port if different.

curl -s http://localhost:5000/api/substances | jq '.[0]'
curl -s http://localhost:5000/api/foods | jq '[.[].simulants]'
curl -s http://localhost:5000/api/foods/1 | jq

Tip: pipe responses through jq for quick inspection, or fetch from your application client with the same URLs.
